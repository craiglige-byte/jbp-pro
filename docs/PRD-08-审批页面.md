# PRD-08：审批页面

> JBP Pro — 联合生意规划助手 · 附加模块
>
> 版本: V1.1 | 日期: 2026-07-15

---

## 1. 模块概述

年度生意计划归档后，审批人（省区渠道+省总）通过审批链接查看经销商年度计划详情，并进行审批操作。

## 2. 功能需求

### 2.1 页面展示

- 基于预览计划生成只读 HTML，无 header/步骤条/编辑按钮
- 数据锁定为归档时的快照，不可修改
- 页面底部放置白底审批卡片（非悬浮），宽度与上方内容一致，底部圆角

### 2.2 审批卡片

| 元素 | 说明 |
|------|------|
| 文案 | 「请仔细审阅以上年度计划内容，完成审批。」 |
| 通过按钮 | 绿色，点击弹出确认弹窗 |
| 驳回按钮 | 红色，点击弹出驳回原因弹窗 |

### 2.3 60 秒倒计时

- 后台计时，不在页面上显示
- 60 秒内点击通过或驳回，顶部弹出黄色提示：「请仔细研读年度计划，阅读时长建议不少于一分钟。」
- 60 秒后正常执行审批操作

### 2.4 通过审批

- 点击通过 → 满足 60s → 弹出确认弹窗
- 弹窗标题：「通过审批」
- 弹窗内容：「确认通过此年度生意计划审批？」
- 按钮：取消（灰色）+ 确认通过（绿色）
- 确认后，审批卡片替换为绿色结果卡片：「✓ 审批结果：已通过」

### 2.5 驳回审批

- 点击驳回 → 满足 60s → 弹出驳回弹窗
- 弹窗标题：「驳回审批」
- 驳回原因：必填，200 字以内，带字数统计
- 按钮：取消（灰色）+ 确认驳回（红色）
- 确认后，审批卡片替换为：「已驳回当前审批」+「驳回备注：xxx」

### 2.6 弹窗样式

- 通过弹窗和驳回弹窗结构统一：左对齐标题和内容，右下角按钮组
- 弹窗宽度 max-width:440px，圆角 16px
