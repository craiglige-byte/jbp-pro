# PRD-08：审批页面

> JBP Pro — 联合生意规划助手 · 附加模块
>
> 版本: V1.0 | 日期: 2026-07-08

---

## 1. 模块概述

年度生意计划归档后，审批人（省区渠道+省总）通过审批链接查看经销商年度计划详情，并进行审批操作。

## 2. 功能需求

### 2.1 页面展示

- 基于预览计划生成只读 HTML，无 header/步骤条/编辑按钮
- 数据锁定为归档时的快照，不可修改
- 页面底部固定显示审批操作栏

### 2.2 审批操作栏

| 元素 | 说明 |
|------|------|
| 倒计时 | 60 秒倒计时，从页面打开时开始计时 |
| 通过按钮 | 倒计时归零前灰色不可点击，归零后变为绿色可点击 |
| 驳回按钮 | 倒计时归零前灰色不可点击，归零后变为红色可点击 |

### 2.3 业务规则

- 审批人必须等待 60 秒（确认已阅读完整计划）后才能操作
- 点击「通过」→ 确认弹窗 → 提交审批通过
- 点击「驳回」→ 确认弹窗（可填写驳回原因）→ 提交审批驳回
- 操作后按钮区域显示审批结果状态

### 2.4 交互流程

```
打开审批链接 → 页面加载 → 倒计时 60s 开始
                                    ↓
                              倒计时归零，按钮激活
                                    ↓
                      ┌─────────┴─────────┐
                      ↓                   ↓
                  点击通过              点击驳回
                      ↓                   ↓
                 确认弹窗              驳回原因弹窗
                      ↓                   ↓
                「审批通过」          「审批驳回」
```
