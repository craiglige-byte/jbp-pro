# PRD-07：预览计划

> JBP Pro — 联合生意规划助手 · 模块 7/7
>
> 对应组件：`components/ReviewStep.tsx`
> 对应数据分区：全部 `JBPData`（全量回看）
> 对应 WizardStep：`'review'`

---

## 1. 模块概述

预览计划是 JBP 规划的最后一步，将所有前续步骤的数据进行全量回看和最终确认。本模块以卡片摘要 + 可展开详情的方式展示从基本信息到规划预算的完整规划内容，支持打印和导出 HTML。

### 1.1 核心功能清单

| 序号 | 功能 | 说明 |
|------|------|------|
| F01 | 基本信息回看 | 经销商名称、负责人、规划周期、授权区域概要 |
| F02 | 经营回顾回看 | 可弹窗查看完整的 BusinessReviewStep |
| F03 | 目标回看 | 4 个目标及其目标值摘要 |
| F04 | 策略回看 | 每个目标下的策略列表 + 衡量标准 |
| F05 | 拆解计划回看 | 弹窗查看进货/销售/库存/盈利拆解详情 |
| F06 | 行动回看 | 行动项列表 + 按负责人筛选 + 季度分布 |
| F07 | 预算回看 | 弹窗分 Tab 查看 5 类预算明细 |
| F08 | 未完成步骤提示 | 高亮标记未完成的步骤，提醒用户完善 |
| F09 | 打印样式 | `@media print` 适配打印输出 |
| F10 | 归档审批 | 弹窗展示经销商生意计划状态（正常/未生成草稿/发生变更）；存在变更时不可归档 |
| F11 | 关闭详情 | 经营回顾弹窗右下角黑色按钮关闭弹窗 |
| F12 | 拆解目标 | 策略与行动表格，无衡量(Measure)列 |

## 3. 打印样式

### 3.1 打印适配规则

```css
@media print {
  /* 隐藏交互元素 */
  button, .no-print { display: none !important; }
  
  /* 移除背景色 */
  body { background: white !important; }
  
  /* 展开所有折叠区域 */
  .collapsed-content { display: block !important; }
  
  /* 卡片边框适配 */
  .card { border: 1px solid #e2e8f0; break-inside: avoid; }
  
  /* 全宽显示 */
  .max-w-7xl { max-width: 100% !important; }
}
```

### 3.2 打印内容

打印时自动展开所有折叠区域，隐藏按钮、导航和 Footer。

---

## 4. 导出 HTML

点击「导出 HTML」按钮时：
1. 生成独立的 `.html` 文件，包含完整的 JBP 数据
2. 内嵌所有 CSS（Tailwind 编译后的样式）
3. 静态化 Recharts 图表（转为 HTML table 或 SVG inline）
4. 文件名格式：`JBP_Plan_{经销商名}_{周期}_FY.html`
5. 自动触发浏览器下载

---

## 5. 导航规则

- 进入条件：`isActionStepComplete()`
- 离开条件：无（最后一步，仅能返回上一步）
- 上一步：`'budget'`
- 下一步：无
