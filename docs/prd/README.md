## PRD 间依赖关系

```
PRD-01 基本信息
  ↓
PRD-02 经营回顾  ──── 数据基线输入
  ↓
PRD-03 设定目标  ──── 依赖经营回顾数据
  ↓
PRD-04 拆解策略  ──── 依赖目标数据
  ↓                    ├─ 进货拆解 → 库存拆解
  ↓                    └─ 盈利拆解 → 预算模块
PRD-05 落实行动  ──── 依赖策略数据
  ↓
PRD-06 规划预算  ──── 依赖行动数据 + 盈利拆解
  ↓
PRD-07 预览计划  ──── 依赖全部数据
```

---

## 修改约定

- 每个模块的功能变更 **只更新对应的 PRD 文件**
- 跨模块变更（如数据类型修改）需同时更新受影响的 PRD
- 每次修改 PRD 时更新文件末尾的「修改日志」表
- PRD 版本号规则：主版本.次版本（V1.0 → V1.1 → V2.0）
